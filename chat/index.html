채팅

채팅에 참여하여 사람들과 대화하거나 질문 및 답변을 할 수 있습니다.

채팅 서비스 선택

Discord, Matrix (Element.io), IRC (Libera.chat) 로 채팅에 참여하실 수 있습니다. 아래 정보를 참고하여 본인이 사용하기 편한 방식으로 이용해 주세요.

  • Discord: Discord 서버 들어가기 | Discord 로그인
  • Matrix (Element.io): Matrix Space 접속 (#ubuntu-kr) | Element.io 로그인 | Element.io 회원가입
  • IRC (Libera.chat): irc.libera.chat 의 #ubuntu-kr 채널 접속. (웹에서 Kiwi IRC로 바로 접속 하기)

IRC 이용정보

사용하시는 IRC 클라이언트가 있다면, 아래 정보로 IRC 채널에 접속하여 이용할 수 있습니다.

  • 서버: irc.libera.chat (또는 irc.ubuntu.com - irc.libera.chat 을 가리키는 주소)
  • 포트
    • 평문 통신: 6665~6667 또는 8000~8002
    • TLS 암호화 통신: 6697, 7000, 7070
  • 채널: #ubuntu-kr
  • IRC 이용 정보 자세히 보기

IRC 이용시, 아래 규정을 먼저 확인 해 주시기 바랍니다.

  • IRC 이용 규정
  • Ubuntu IRC 채널 이용약관

채널 목록

아래 채널 목록 표를 참고하여 이용해 주시기 바랍니다. 일부 채널은 서로 다른 메신저 서비와 연동 되거나, 대화 내용이 기록되고 있습니다.

채널 용도 Discord Matrix (Element.io) IRC (Libera.chat) 비고
일반적인 대화 및 질의응답 general #loco-kr-general:ubuntu.com #ubuntu-kr IRC 대화기록 확인하기*
공지사항 채널 notice #loco-kr-notice:ubuntu.com - -
번역(L10n)팀 채널 l10n #l10n-ko:ubuntu.com - -
UbuCon Korea 참가자 채널 ubucon-kr #ubucon-kr:ubuntu.com - -
포럼 게시물 피드 채널 forum-feed #loco-kr-forum-feed:ubuntu.com - -

*IRC 의 #ubuntu-kr 채널에서의 대화 내역은 Canonical IS 팀이 운영하는 ubuntulog 봇에 의해 모두 기록되고 있으며, irclogs.ubuntu.com 에서 IRC 대화 기록을 확인하실 수 있습니다. 또한, 연동된 다른 메신저의 채널에도 IRC 관련 규정이 적용되니 이용에 참고 해 주시기 바랍니다.
